Privacy Policy

Summary

  • FollowDiff does not collect any personal data

Overview

FollowDiff is a Chrome browser extension designed to help users track changes in Instagram follower and following lists. This Privacy Policy explains how information is handled when using the extension.

The extension operates entirely within your web browser. When you use FollowDiff, it performs the following functions:

  • Retrieves follower and following data from Instagram using your existing browser session
  • Stores snapshot data locally using Chrome's built-in storage API
  • Compares snapshots to display changes in follower and following lists

All operations occur locally on your device. The extension does not communicate with any external servers operated by the developer.

Information Collection

FollowDiff does not collect, transmit, or store any user information on external servers. The developer does not operate any servers, databases, or analytics services in connection with this extension.

Specifically, the extension does not collect:

  • Personal identification information
  • Instagram account credentials
  • Browsing history or activity
  • Device information or identifiers
  • Location data
  • Any other personally identifiable information

No Data Collection

The extension architecture does not include any mechanism for transmitting data to external servers. All functionality operates entirely within your browser environment.

Local Data Storage

All data generated by FollowDiff is stored locally on your device using Chrome's chrome.storage.local API. This data includes:

  • Saved follower and following snapshots
  • Profile usernames associated with snapshots
  • Extension configuration settings

This data remains on your device and is only accessible to you. If you uninstall the extension, Chrome automatically removes all associated stored data.

Instagram Data Access

When retrieving follower data, the extension makes requests directly from your browser to Instagram's servers. These requests utilize your existing Instagram session cookies—the same authentication your browser uses when you visit instagram.com directly.

The extension does not:

  • Store or access your Instagram password
  • Store Instagram authentication tokens
  • Route requests through proxy servers
  • Modify your Instagram account settings or content

The extension performs read-only operations and cannot make any modifications to your Instagram account.

Third-Party Services

FollowDiff does not integrate with or utilize any third-party services. This includes:

  • Analytics platforms (such as Google Analytics or similar services)
  • Crash reporting services
  • Advertising networks
  • External APIs or data services

Browser Permissions

The extension requires the following Chrome permissions to function:

  • storage — Required to save snapshots and settings locally on your device
  • activeTab — Required to interact with the current Instagram tab when using the "Fetch from current tab" feature
  • tabs — Required to locate an Instagram tab for data retrieval
  • Host permission for instagram.com — Required to execute the content script that retrieves follower data

These permissions represent the minimum required for the extension to operate. The extension does not request broad browsing data access or permissions for websites other than Instagram.

Data Export

The extension provides functionality to export snapshot data. Exported files are saved directly to your device as JSON or text files. The developer has no access to exported data.

Children's Privacy

FollowDiff is not intended for use by individuals under the age of 13. The extension does not knowingly collect information from children, as it does not collect information from any users.

Policy Updates

This Privacy Policy may be updated periodically to reflect changes in the extension or applicable regulations. As the extension updates automatically through Chrome, users will always have access to the current version. Material changes to this policy will be noted in the extension's update notes.

Contact Information

For questions regarding this Privacy Policy or the extension, please contact the developer through the Chrome Web Store listing or the project's GitHub repository.

Privacy by Design

FollowDiff was developed with privacy as a core principle. The architecture ensures that no user data leaves your browser, providing complete control over your information.
